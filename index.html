How to play: Use your arrow keys to move the tiles. When two tiles with the same school touch, they merge into one!
Note: The order of schools is not intended to be a ranking (It was generated through random.org).

Created by Elbert Ho. Inspired by Get MIT by Mitch Gu. Based on 2048 by Gabriele Cirulli. and 1024 by Veewo Studio and conceptually similar to Threes by Asher Vollmer.

Number reference: 2 = Delbarton, 4 = Morristown Beard, 8 = Newark Academy, 16 = Kent Place, 32 = Peck, 64 = Princeton Day, 128 = Rutgers Prep, 256 = Gill St. Bernards, 512 = Montclair Kimberly (MKA), 1024 = Primoris, 2048 = Pingry
